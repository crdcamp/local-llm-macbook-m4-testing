# The Purpose

I want a local LLM research assistant that runs smoothly on **Apple M4 silicone**. I'll make a bunch of tools and stuff like that for it to work well. I think AI is best used for gathering information and presenting it to you. It's up to the user to actually investigate and understand the desired material. 

So, if I can find a model that runs smoothly on Apply M4 Silicone through this testing, I might not need to pay for AI subscriptions anymore.

Thus, this project benchmarks a bunch of local models in preparation for that application. While these benchmarks (according to my limited knowledge) are probably not sufficient to make any absolute claims, this will still provide some decent data regardless.
